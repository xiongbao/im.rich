## 🎨 设计理念

> Success is just failure with better typography.

- **排版即哲学**：留白比内容重要，字距比字义重要
- **反差即幽默**：越正经的外壳，越荒诞的内核
- **克制即奢侈**：不用一个多余的颜色，不写一行多余的 JS
